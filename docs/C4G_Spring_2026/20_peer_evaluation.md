# Peer Evaluation Instructions
This page provides helpful resources for downloading and running a standalone version of the Spring 2026 version of the BLIS software.

Troubleshooting contact: Haki Atalov (hatalov3@gatech.edu) or Ranjini Unnikrishnan (runnikrishnan6@gatech.edu) on Microsoft teams

## System Requirements
1. Windows 10 or later
2. x86/x64 CPU

## Downloading BLIS
1. Please download our application artifact here: [BLIS Standalone Zip File](https://github.com/C4G/BLIS/releases/tag/v4.0.0.beta1)
2. Once downloaded, extract the contents of the file. 
3. Open the extracted folder and find the `BLIS-NG.exe` launcher. 
4. Run the `BLIS-NG.exe` launcher. This will start the BLIS instance in your browser. 
5. Use the following credentials to log in:
    - Username: `testlab1_tech1`
    - Password: `tech123`

## Peer Review Survey
The survey will ask the user to complete three tasks. The survey will provide detailed task information, but at a high level the tasks relate to the following BLIS functions:  

1. Register a new patient and subsequently register a new specimen. 
2. Reset the user password.
3. Initiate a BLIS software update (UI only).
